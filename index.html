Hi there! We build projects for learning to code, like the atto fantasy computer

We wanted to put the instant excitement of seeing your own code run for the first time back into programming. Using a professional development environment can be unappealing to younger learners, and too complex for first-time coders. That's why we built atto, our fantasy computer and programming language that you can get to grips with in seconds.
